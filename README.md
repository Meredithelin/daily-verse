# 诗笺 · 每日一言

每天一句名人名言或唐诗宋词的网页应用。中式宣纸美学，单文件、可离线（PWA），可挂 GitHub Pages。

## 功能

- **今日一条**：按日期确定性选取，当天刷新结果不变、跨设备一致；午夜自动换新。
- **三类内容**：名言（中/外）· 诗词（唐诗宋词，含译文 + 赏析）· 名句（千古名句）。
- **换一条 / 上一条·下一条**：在当前分类里随机或顺序浏览。
- **竖排**：诗词可切换古籍式右起竖排。
- **收藏**：心仪的存入本地，「收藏」标签随时回看。
- **复制 / 分享**：一键复制规整文案，移动端调用系统分享。
- **深色「夜读」模式**：跟随系统或手动切换，偏好本地记忆。

内容共 135 条（30 首唐诗宋词 + 69 条名言 + 36 句名句），全部内联在 `index.html`，无需联网。

## 本地运行

```bash
cd ~/daily-verse
python3 server.py          # 然后打开 http://localhost:8792
# 或： python3 -m http.server 8792
```

## 文件

| 文件 | 说明 |
| --- | --- |
| `index.html` | 应用本体（含全部内容数据，单文件即可运行） |
| `manifest.webmanifest` / `icon.svg` | PWA 配置与图标 |
| `sw.js` | Service Worker，离线缓存 |
| `poems_data.json` | 唐诗宋词源数据（供构建脚本用） |
| `build_data.py` | 把内容合并注入 `index.html` 的构建脚本 |
| `server.py` | 本地预览服务器 |

## 维护

- **改内容**：编辑 `poems_data.json` 或 `index.html` 里 `DATA:BEGIN/END` 之间的数据。
- **改版后**：把 `sw.js` 里的 `CACHE` 版本号 +1（如 `shijian-v1` → `shijian-v2`）再发布，用户才能拿到更新。
